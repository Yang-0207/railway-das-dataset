# 铁路周界上缆分布式光纤传感（DAS）标准化数据集说明

本数据集是铁路周界现场采集的上缆分布式光纤传感（DAS）标准化数据集，完成了栅栏清洗后的全流程数据清洗与事件标注，专为铁路周界安防场景下的入侵事件识别、环境干扰区分与智能监测算法研究构建。

## 1. 数据规格与时空特性

- **单样本维度**：单条数据样本为 `3001 × 5` 维度的 DAS 振动信号矩阵
  - **时间维度**：3001 个时间采样点对应 6 秒的连续监测时长，对应系统采样率约为 500 Hz（`3001 / 6 ≈ 500 Hz`），匹配分布式光纤传感系统的常规工程配置
  - **空间维度**：5 个空间采样通道对应 25 米的光纤监测段，单通道覆盖 5 米监测范围，为沿铁路栅栏上部布设的上缆周界监测区域

- **数据本质**：DAS 系统采集的光纤沿线振动 / 应变信号，可精准捕捉周界结构的微小扰动，实现不同类型事件激励的差异化表征。

## 2. 事件类别体系

数据集按事件类型划分为 7 个类别，全面覆盖铁路周界的环境干扰、常规运营事件与入侵 / 破坏风险事件。

| 类别名称 | 事件类型 | 场景说明 |
|---|---|---|
| 背景 | 环境本底信号 | 无人工扰动的铁路周界日常环境噪声，如环境微振动、系统背景噪声等 |
| 过车 | 运营背景事件 | 列车通过周界监测段时产生的光纤振动信号，为铁路运维场景下的常规背景事件 |
| 风 | 气象干扰事件 | 自然风作用于栅栏 / 光纤产生的振动信号，是周界安防中典型的环境干扰源 |
| 落石 | 地质灾害事件 | 落石撞击周界栅栏产生的振动信号，对应铁路沿线边坡落石等地质灾害风险 |
| 敲击 | 入侵试探事件 | 人工敲击栅栏的模拟入侵动作，为周界入侵的典型前置试探行为 |
| 攀爬 | 主动入侵事件 | 人工攀爬栅栏的入侵行为，是铁路周界安防的核心监测目标 |
| 拆卸 | 设施破坏事件 | 人工拆卸栅栏的破坏行为，对应周界物理设施的安全风险 |

## 3. 数据集应用价值

该数据集可用于以下方向的研究与应用：

- 铁路周界入侵事件识别
- 环境干扰与真实风险事件区分
- DAS 智能监测算法开发与评估
- 周界安防场景下的多类别事件检测与分类
- 铁路沿线灾害与设施破坏风险预警
